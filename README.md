## <u>**Projet JSF de la gestion des employés d'une entreprise*</u>
application pour la gestion des employés dans une entreprise. L'objectif est de créer une application web avec JSF et Hibernate permettant de gérer les services et les employés de l'entreprise

## <u>**Structure du projet du Projet**</u>
la structure du projet et comme suite :

+ `/src/java`  : Contient les sources Java du projet.
+ `/web/ressources`  :  Contient les fichiers de ressources, tels que les fichiers de configuration.
+ `/web/WEB_INF`  : Contient les fichiers de configuration spécifiques à l'application.
+ `/lib` : Répertoire pour les dépendances du projet (les bibliothèques JAR).

## <u>**les fonctionnalités implémentées**</u>
1. les operations CRUD pour les employes.
2. les opérations CRUD pour les services.
3. Recherche des employés par service.
4. Recherche des employés entre deux date.
5. Grpahe nombre d'employé par service
6. Grpahe nombre d'employé par chef.
